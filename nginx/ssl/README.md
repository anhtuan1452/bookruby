# SSL Certificates
Đặt các file certificate từ Cloudflare vào đây:
- cert.pem (Cloudflare Origin Certificate)
- key.pem (Private Key)

## Cách lấy Cloudflare Origin Certificate:
1. Đăng nhập vào Cloudflare Dashboard
2. Chọn domain của bạn
3. Vào SSL/TLS → Origin Server
4. Click "Create Certificate"
5. Chọn các options phù hợp và tạo certificate
6. Copy nội dung Certificate và Private Key vào các file tương ứng
